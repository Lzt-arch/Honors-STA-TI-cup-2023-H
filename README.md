# Honors-STA-TI-cup-2023-H
2023年TI杯电子设计竞赛H题方案开源，作品来源HDU卓越科协
	这是2023年电子设计竞赛H题信号分离的总结，使用了两块STM32和一块FPGA，两块DAC904，一块AD630，一个偏置模块，一块4.3寸串口屏，TPS5450+LDO的电源塔，OPA227加法器

1.主要思路和解题步骤：

	由信号源产生两路题目要求的波，经过增益为1的加法器直接送入第一块，STM32（STM32F407VET6），同时送入AD630锁相放大器的其中一个输入端，F4将相加波做FFT运算，找到频谱中两个幅度最大的频率就是两输入波的频率，进行波形判断（3ω、5ω法），再通过usart1控制串口屏显示相关信息，波特率为115200，然后通过另一路串口与FPGA通信，传递”低频波频率，低频波波形，高频波频率，高频波波形，相位差“，波特率为9600，其中相位差是控制高频率做分辨率为5°的相移（发挥部分2）

	FPGA（Cyclone4 EP4CE10F17I7）接受五组数据作为DAC的数据，DAC模块使用的是14位DAC904（凌智），寻址范围升级到了48位，频率分辨率可以达到的分辨率为165MHz/2^48≈0.000000586Hz , 即0.586 uHz，在完成发挥部分2中通过：（1）两路DAC使用同一个165M时钟源，（2）每次有参数改变时都会产生同步清零信号：sync_rst 脉冲，两个 DAC 的相位累加器会被在同一个时钟周期内强制清零。这意味着两路波形永远都是从 0°（原点）同时起跑的。如果没有这个机制，改变频率时两路波形的相对相位就是随机的，示波器上就会看到波形乱跑，无法锁定。这两个约束确保了两路DAC可以在示波器上同频显示不漂移，此外为两路DAC设置了一个精密的步进K_STEP = K_STEP_BASE + fine_tune_ext;，用于相位移动。

	在FPGA的调试过程中发现，当两路DAC中的高频波是低频波的偶数倍时，它们的初相位不是0°，而是180°，这个由于没有找到原因，所以增加了对于这种情况，使高频波自动相移180°作为初始相位的约束
	
	接着两路DAC过一个增益为1的加法器送入AD630的另一端（REF），由另一块STM32（STM32H743VIT6）进行ADC采样输出的直流信号，众所周知，两路相加信号的相位差越小，AD630输出的直流信号越大（输出范围为-1V~1V），由于AD630可能输出负压，所以H7采样之前接了一个电压偏置模块抬升1V，当当前采样值相比上一次采样值增大，说明相位差正在减小，将PB12=1, PB13 打一个脉冲，否则PB12=0, PB13 打一个脉冲，这两个引脚分别接FPGA的N9和T10，PB12为1时，FPGA以2000*0.586 uHz/每时钟周期 的速率减小频率，PB12为0时，FPGA以2000*0.586 uHz的速率增大频率，当相位差相差不多，PB12在0/1直接来回跳动3次时，变换速率降为600*0.586 uHz/每时钟周期进行微调跟随，保证A和A`，B和B`能够同频显示，实测基本在10s内完成收敛，稳定不漂移。

仍然存在的问题：
	
	三角波判不准，高频判不出来，低频来回跳动，A和A`，B和B`的同频显示偶尔会出现锁不住的情况，还会又过一段时间失锁的现象。

额外功能：

	对于发挥部分2，可以做到5kHz为倍数，一直到几兆都可以同频显示和相移（ch1和ch2成倍数时）
