	# 2023年全国大学生电子设计竞赛H题 信号分离项目总结
## 一、硬件整体配置
主控与信号处理芯片：2块STM32（F407VET6、H743VIT6）+ 1片Cyclone4 FPGA（EP4CE10F17I7）
信号通路器件：2片DAC904（14位高速DAC）、AD630锁相放大器、OPA227加法器、电压偏置模块
人机交互：4.3寸串口显示屏
电源系统：TPS5450开关电源 + LDO多级稳压电源塔

注意：第二版不同频率相位相同工程为最终的FPGA工程，VET6FFT是F4的工程，H7_ADC是H7的工程，H7在本装置中发挥的作用很小，用F1都可以实现，主要是为了练习H7才选择使用的

## 二、主要思路与完整解题步骤
1. **前端信号采集与频谱分析（STM32F407）**
    信号源输出两路待测波形，经增益1的OPA227加法器混合后分两路输出：
    - 一路送入STM32F407 ADC；
    - 另一路接入AD630锁相放大器输入通道。
    F4芯片对混合信号执行FFT频谱运算，提取频谱幅值最大的两个峰值频率，区分高低频分量；采用3ω、5ω判据完成波形类型识别。
    - 串口屏通信：USART1，波特率115200，实时显示波形频率、波形类型、相位差等参数；
    - FPGA数据交互：另一路串口，波特率9600，向下位机传输5组关键参数：低频频率、低频波形类型、高频频率、高频波形类型、目标相位差，用于实现发挥部分5°分辨率相位调节。

2. **双通道高精度波形生成（FPGA）**
    FPGA接收STM32下发参数，驱动两片14位凌智DAC904输出两路标准波形：
    - 频率精度设计：DAC相位累加器寻址位宽扩展至48位，系统时钟65MHz，理论频率分辨率165MHZ/2^48=0.000000586Hz
    - 同步锁相机制（解决波形漂移）：
      1. 两片DAC共用同一路165MHz系统时钟；
      2. 参数更新时统一输出`sync_rst`同步清零脉冲，两路DAC相位累加器在同一时钟周期同步复位至0°初始相位，保证波形起点对齐，避免示波器波形随机漂移。
    - 相位步进控制：设计精密相位步进参数 `K_STEP = K_STEP_BASE + fine_tune_ext`，实现最小5°相位步进调节。
    - 偶数倍频相位补偿BUG修复：调试发现当高频为低频偶数倍时，两路波形天然存在180°固有相位偏移，程序增加自动补偿逻辑，此类场景下高频波形初始相位强制偏移180°。

3. **锁相闭环相位跟踪（STM32H743 + AD630）**
    FPGA输出两路波形经另一路增益1加法器混合后，送入AD630参考输入端（REF），构成锁相闭环：
    - AD630输出特性：两路信号相位差越小，输出直流电压幅值越大，输出范围 `-1V ~ +1V`；
    - 负压采样预处理：增加电压偏置模块，将AD630输出抬升1V，使电压全部转为正电压，适配STM32H7单极性ADC采样；
    - 相位差判别逻辑：对比本次与上一次ADC采样值：
      - 采样值变大 → 相位差减小：`PB12=1`，`PB13`输出脉冲；
      - 采样值变小 → 相位差增大：`PB12=0`，`PB13`输出脉冲；
    - 引脚硬件互联：`PB12`、`PB13`分别连接FPGA引脚N9、T10，用于传递相位调节方向触发信号；
    - 自适应变步长锁相：
      1. 粗调阶段：单步调节速率 `2000 × 0.586 μHz`；
      2. 接近锁定阶段：当`PB12`电平0/1来回跳动3次，切换为细调速率 `600 × 0.586 μHz`，精细跟随相位；
    - 闭环指标：实测系统收敛时间约10s，锁定后波形稳定无明显漂移。

## 三、现存待优化问题
1. 三角波波形识别算法精度不足，易误判；
2. 高频分量频谱提取不稳定，部分工况无法识别高频信号；
3. 低频频率输出存在持续小幅跳变；
4. 同源波形A/A'、B/B'同步显示偶发失锁；
5. 长时间稳定运行后会出现周期性失锁漂移现象。

## 四、拓展额外功能（发挥部分2）
支持两路波形成整数倍频关系场景下的同步输出与高精度相移调节：以5kHz为步进基准，覆盖从低频至数MHz全频段，均可实现示波器双通道同步稳定显示、5°步进相位偏移控制。
	对于发挥部分2，可以做到5kHz为倍数，一直到几兆都可以同频显示和相移（ch1和ch2成倍数时）
